# Aura - 腦波監測應用更新日誌

## [1.1.0] - 2025-10-15

### ✨ 新增功能
- **Mind Link (NeuroSky) 裝置支援** - 添加對 NeuroSky 腦波裝置的完整支援
- **雙重裝置系統** - 同時支援 BLE 和藍牙經典裝置
- **ThinkGear 協定解析** - 實現完整的 NeuroSky ThinkGear 數據協定解析
- **實時腦波數據處理** - 專注度、放鬆度、信號品質實時監測
- **增強的設備管理** - 改進的設備列表界面，分類顯示不同類型裝置

### 🔧 技術改進
- **MindLinkManager.swift** - 新增專門的 Mind Link 裝置管理器
- **ExternalAccessory 框架整合** - 支援藍牙經典裝置連接
- **數據安全處理** - 防止算術溢出和記憶體洩漏
- **錯誤處理機制** - 完善的連接狀態管理和錯誤恢復
- **性能優化** - 限制大數據包處理，避免 UI 卡頓

### 🎨 界面更新
- **設備列表頁面** - 分別顯示 BLE 和 Mind Link 裝置
- **設置頁面** - 顯示兩種連接類型的狀態
- **連接狀態指示** - 更清晰的裝置連接狀態顯示

### 📋 配置要求
- **Info.plist 配置** - 需要添加 `UISupportedExternalAccessoryProtocols`
- **框架依賴** - 新增 `ExternalAccessory.framework` 依賴

### 🐛 修復問題
- 修復校驗和計算的算術溢出問題
- 改進數據流處理的穩定性
- 優化記憶體使用和資源管理

### 📱 使用說明
1. 在 iOS 設定中配對 Mind Link 裝置（配對密碼：0000）
2. 確保系統藍牙顯示「已連接」
3. 在應用的設備頁面選擇並連接 Mind Link
4. 在數據和波形頁面查看實時腦波資料
